GET STARTED

Build it from source

WebToApp is a standard Gradle Android project. Clone the repo, open in Android Studio (Hedgehog or newer), and run on any device with API 23+. Or run the Gradle wrapper directly to assemble a debug APK.

View on GitHub 
# Clone
git clone https://github.com/shiahonb777/web-to-app.git
cd web-to-app

# Build a debug APK
./gradlew assembleDebug

# Output
app/build/outputs/apk/debug/*.apk
